## Gekozen API
Voor dit experiment gebruik ik de Chuck Norris Joke API:
https://api.chucknorris.io/jokes/random

Deze API geeft een willekeurige grap terug in JSON-formaat.

## Wat doet mijn script?
- Stuurt een GET-request naar de API
- Ontvangt een JSON-response
- Toont de statuscode, de joke ID en de grap zelf

## Hoe voer je het script uit?
In de terminal:

python3 ap3_chucknorris.py
